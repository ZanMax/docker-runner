# Docker Runner

Run different prepared docker containers from UI / console

## Recipes
You can configure your own recipes in `config.json` file.

- MySQL      	      
- Postgres   	      
- Mongo
- Redis
- RabbitMQ
- Memcached

## Additional recipes
You can add additional recipes to the `recipes` folder. 

- Gitlab
- Jenkins
- Pi-hole
- Wireguard
